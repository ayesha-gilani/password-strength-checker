# 🔐 Password Strength Checker

A Python-based Password Strength Checker developed as part of my DecodeLabs Cyber Security Internship. This application analyzes a user's password based on multiple security criteria and classifies it as Weak, Medium, or Strong while providing recommendations for improvement.

---

## Features

- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Detects commonly used weak passwords
- Calculates a password security score
- Classifies passwords as:
  - Weak
  - Medium
  - Strong
- Provides suggestions to improve password security

---

## Technologies Used

- Python 3
  
---

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ayesha-gilani/password-strength-checker.git
```

2. Open the project folder

3. Run

```bash
python password_strength_checker.py
```
